 Análisis de Ventas - Proyecto de Data Analyst

Este proyecto tiene como objetivo analizar un dataset de ventas para obtener insights relevantes sobre el rendimiento del negocio.

Se trabajó con Python utilizando la librería pandas para la limpieza, transformación y análisis de los datos, así como matplotlib para la visualización.

El análisis se enfoca en identificar patrones de ventas, productos más demandados y comportamiento general del negocio.

Tecnologías utilizadas

- Python  
- pandas  
- matplotlib  
- Guoogle Colab  

El proyecto se desarrolló en las siguientes etapas:

### 1. Carga de datos
- Importación del dataset desde archivo CSV/Excel

### 2. Exploración
- Revisión de tipos de datos
- Detección de valores nulos
- Análisis inicial de la estructura

### 3. Limpieza
- Conversión de fechas
- Eliminación de valores nulos
- Preparación de datos para análisis

### 4. Transformación
- Creación de nuevas variables (mes, año)
- Preparación de datos para agrupaciones

### 5. Análisis
Se respondieron preguntas clave como:
- ¿Cuáles son los productos más vendidos?
- ¿Cómo evolucionan las ventas en el tiempo?
- ¿Qué categorías generan más ingresos?
- ¿Qué clientes aportan mayor volumen de ventas?

### 6. Visualización
- Gráficos de ventas por mes
- Top productos
- Ventas por categoría

---

## Principales insights

- Se identificaron los productos con mayor volumen de ventas.
- Se observaron patrones temporales en la evolución de las ventas.
- Algunas categorías presentan mayor impacto en los ingresos totales.
